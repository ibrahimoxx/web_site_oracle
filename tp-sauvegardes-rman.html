Resume TP sauvegardes avec RMAN

Cette page condense les commandes pratiques du TP: sauvegarde de base complete, PDB, tablespace, backupset, image copy, retention et incremental.

1. Sauvegardes de base

  • REPORT SCHEMA pour voir la structure.
  • Backup de la base complete.
  • Backup avec archived redo logs.
  • Backup d’une PDB.
  • Backup d’un tablespace d’une PDB.

2. Formes de backup

  • BACKUP AS BACKUPSET pour le format backup set.
  • BACKUP AS COPY DATAFILE pour une image copy.
  • Backup d’archivelog en copie selon un motif.

3. Control file et spfile

  • Verifier si CONTROLFILE AUTOBACKUP est active.
  • L’activer si besoin pour proteger la structure de la base.

4. Incremental

  • Activer le block change tracking.
  • Realiser un incremental level 1 apres une base de niveau 0.
  • Executer un recover a partir du tag de copie si le scenario le demande.
